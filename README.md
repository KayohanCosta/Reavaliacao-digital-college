## Lógica com JavaScript

- **Tipos de dados**
  - `string`, `number`, `boolean`, `null`, `undefined`
  - `array`, objeto (object literal)

- **Operadores**
  - Lógicos: `&&`, `||`, `!`
  - Aritméticos: `+`, `-`, `*`, `/`, `%`
  - Comparação: `==`, `===`, `!=`, `!==`, `>`, `<`, `>=`, `<=`
  - Atribuição: `=`, `+=`, `-=`, `*=`, `/=`

- **Variáveis**
  - `var` x `let` x `const`
  - Escopo
  - Hoisting

- **Estruturas de controle de fluxo**
  - Condicionais
    - `if`, `else`, `else if`, `switch case`
  - Loops
    - `for`, `while`, `for in`, `for of`

- **Funções**
  - Funções declaradas
  - Funções *arrow* (`() => {}`)
